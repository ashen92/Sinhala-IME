# Sinhala IME

A free Sinhala Input Method Editor for Windows. Type in English, get Sinhala.

![Getting Started](/assets/demo.gif)

## Features

- **Phonetic typing** — type Sinhala the way it sounds, no new keyboard layout to learn
- **Smart predictions** — word suggestions from a 130,000+ word dictionary
- **Works everywhere** — any Windows app that supports text input
